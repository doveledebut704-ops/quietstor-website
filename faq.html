🔐 Vault & Security
QuietStor uses industry-standard security:
  • AES-256-GCM — the same standard used by banks and governments
  • Unique key per file — derived via HKDF from your device's hardware Keystore
  • Password hashing: PBKDF2 with 600,000 iterations
  • Biometric unlock via Android BiometricPrompt
Keys are stored in your device's Trusted Execution Environment (TEE) or StrongBox — they never leave the hardware.
