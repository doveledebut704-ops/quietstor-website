General

QuietStor is a privacy-first Android file manager with an AES-256 encrypted vault. 100% offline, zero tracking, standard ZIP export. Your files never leave your device.
Yes! Core features are completely free: full file manager, encrypted vault (up to 500 MB), password + biometric unlock, and 4 cleaning tools.

Premium (€1.99 / $1.99 — one-time purchase, no subscription) unlocks unlimited vault storage, ZIP export, Mood Stickers, duplicate finder, WhatsApp cleaner, and auto-backup camera.

Vault & Security

  • Algorithm: AES-256-GCM (chunked streaming, 1 MB/chunk)
  • Key storage: Android Keystore (TEE/StrongBox hardware)
  • Per-file keys: HKDF derivation with unique 32-byte salt
  • Password hashing: PBKDF2 with 600,000 iterations
  • Database: SQLCipher with derived passphrase
Your encrypted data is permanently lost. This is by design (zero-knowledge):
  • We never see your password
  • There's no "Forgot Password" option
  • No backdoor, no recovery possible
Tip: Choose a password you'll remember, and create a ZIP backup (Premium) stored safely on your PC.

Storage Guardian

Storage Guardian is the smart cleaning hub with 6 tools: large files, old screenshots, empty folders, APKs (free) + duplicates and WhatsApp (Premium). Files are classified by Trust Gradient (Safe / Review / Sensitive).

Export & Backup

Premium users can export their vault as a standard encrypted ZIP file protected by a strong password (8+ characters, at least 1 letter + 1 digit — different from your vault password).

The ZIP can be opened with 7-Zip, WinRAR, macOS Finder, or any tool that supports AES-256 ZIP encryption. No proprietary format, no lock-in.
1. Export vault as ZIP on old phone → 2. Transfer ZIP (USB, SD card, etc.) → 3. Install QuietStor on new phone → 4. Import ZIP → 5. Enter export password. Done.

Data & Privacy

No. Zero data collection. No analytics, no accounts, no telemetry. Optional opt-in Crashlytics for bug reports only — disabled by default.
